Ant colony

You create a maze, place an ant colony (the center) and several food sources with adjustable nutritional values on it, and specify the number of ants that will explore the maze, find food sources and optimize routes using pheromone trails, taking into account the path length, the nutritional value of the source and the current load on the route; when you run the model, you will see an animation demonstrating the movement of ants, the formation and strengthening of pheromone trails, as well as the adaptation of routes in real time, which will allow you to observe how ants find a balance between various factors to achieve maximum efficiency.
