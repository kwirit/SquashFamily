Clustering algorithm

The feature allows you to interactively place points on a visual plane and automatically group them into clusters using different clustering algorithms: K-Means, K-Means++ and DBSCAN. To get started, simply left-click on the desired location on the plane to add points. Then select the clustering algorithm from the drop-down menu and configure the parameters: for K-Means and K-Means++, specify the number of clusters (k), and for DBSCAN, specify the maximum distance between neighboring points (epsilon) and the minimum number of neighbors (MinPts). Once the parameters are configured, click the "Cluster" button to group the points, which will be highlighted in different colors on the plane. This is a powerful tool for data analysis and visualization, allowing you to easily experiment with different clustering methods.
